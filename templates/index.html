System Overview

Multi-Format Bank Statement Processor with intelligent merchant categorization

This system processes PDF bank statements from various banks and extracts transaction data for analysis and reporting.

Supported Banks: OCBC Bank, DBS Bank, and easily extensible for future banks
Key Features: Automatic format detection, merchant categorization, batch processing
Output Formats: Excel files, CSV data, comprehensive reports

Quick Start Guide

Get up and running in minutes with these simple steps

Step 1: Prepare Your Files

Place your PDF bank statements in the data/ folder. The system supports mixed bank formats and will automatically detect each one.

Step 2: Configure Settings

Go to the Configuration page to set your preferred upload, data, and output folders. These settings apply to both single and batch processing.

Step 3: Process Files

Choose single file processing or batch processing in the Processing page. Single processing is ideal for quick results, while batch processing handles multiple files efficiently.

Step 4: Review Results

Check your configured output folder for processed Excel files, reports, and summaries. All results are organized with clear naming conventions.

Batch Processing Guide

Process hundreds of files efficiently with organized batch processing

How Batch Processing Works

Batch Size: Number of files processed together (e.g., 50 files per batch)
File Selection: Choose specific files from your data folder
Output Options: Create individual batch files, consolidated master file, and summary reports
Progress Tracking: Real-time progress updates during processing

Example Scenario

If you have 200 PDF files and set batch size to 50:

Batch 1: Files 1-50
Batch 2: Files 51-100
Batch 3: Files 101-150
Batch 4: Files 151-200

Plus a consolidated master file containing all 200 files' data.

Troubleshooting

Common issues and solutions for smooth operation

Common Issues

PDF Not Processing

Ensure the PDF is not password-protected and contains text (not scanned images). Check if the format matches supported bank layouts.

Missing Transactions

Verify the PDF format matches supported bank layouts. Some banks may have variations in statement formatting.

Merchant Categories Not Working

Check your merchant configuration syntax in the Configuration page. Format should be: CategoryName = keyword1, keyword2, keyword3

Batch Processing Fails

Verify file permissions and ensure output folder is writable. Check that selected files exist and are accessible.
